# ⏱️ Wall Clock

A modern, minimal, and elegant analog wall clock built using HTML, CSS, and JavaScript.  
This project focuses on clean UI, glassmorphism design, and smooth real-time updates.

---

## ✨ Features

- 🕒 Real-time analog clock (hour, minute, second hands)
- 💻 Digital time display
- 🎨 Modern glassmorphism UI
- 🌙 Dark premium theme
- 📱 Fully responsive design
- ⚡ Lightweight & fast (no libraries)

## ⚙️ How It Works

- JavaScript uses `Date()` object to get current time
- Each hand rotates using CSS `transform: rotate(deg)`
- Clock updates every second using `setInterval()`
